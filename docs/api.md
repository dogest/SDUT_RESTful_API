# API 文档

## 全局状态

- HTTP 200: OK，请求成功
- HTTP 401: 账号认证未通过，请检查账号密码
- HTTP 403: 账号无权限，请确认使用的账号有对应的权限
- HTTP 500: 服务器错误，如果多次出现，请联系开发者解决
